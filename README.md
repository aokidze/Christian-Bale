# Christian Bale Website

Личный сайт, посвященный актёру Кристиану Бэйлу. Сайт выполнен в мрачном стиле "Американского психопата" и "Бэтмена" с элегантными анимациями.

## 📁 Структура проекта

```
Christian Bale/
├── pages/                    # HTML страницы
│   ├── index.html           # Главная страница
│   ├── films.html           # Страница с фильмами
│   └── biography.html       # Биография актера
├── css/                     # Стили
│   └── style.css            # Основной CSS файл
├── assets/                  # Ресурсы
│   ├── images/              # Изображения
│   │   └── ChristianBale.png
│   └── icons/               # Иконки
│       └── Group.svg        # Логотип бэтмена
└── README.md               # Документация
```

## 🎨 Особенности

- **Мрачный дизайн** в стиле "Американского психопата" и "Бэтмена"
- **Золотые акценты** (#FFD700) на черном фоне
- **Плавные анимации** при наведении на карточки фильмов
- **3D эффекты** с rotateX и rotateY
- **Адаптивный дизайн** для всех устройств
- **Квадратные углы** вместо скругленных для жесткого вида

## 📄 Страницы

### index.html (Главная)
- Фото Кристиана Бэйла справа
- Текст и цитата слева
- Навигация в шапке

### films.html (Фильмы)
- Сетка карточек фильмов
- Анимация при наведении с 3D эффектом
- Информация о режиссере и жанре

### biography.html (Биография)
- Полная информация о жизни актера
- Красивые секции с золотыми границами

## 🎬 Анимации

**На странице Films:**
- 3D перспектива (rotateX, rotateY)
- Плавное расширение карточки (0.9s)
- Текст уходит вверх/вниз с translateY
- Заголовок масштабируется (scaleY 1.1)
- Теги сдвигаются вправо с scale эффектом

## 🎨 Цветовая схема

- Основной черный: `#131521`, `#262B42`, `#353B5B`
- Золотой акцент: `#FFD700`
- Темная кровь: `#8B0000`
- Белый текст: `#FFFFFF`

## 📱 Адаптивность

- Полная поддержка мобильных устройств
- Медиа-запросы для планшетов (max-width: 1024px)
- Медиа-запросы для мобильных (max-width: 768px)

## 🚀 Использование

1. Откройте `pages/index.html` в браузере
2. Используйте навигацию для перемещения между страницами
3. Наведите на карточки фильмов для просмотра анимации

## 💻 Технологии

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Google Fonts (Inknut Antiqua)
